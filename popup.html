Setup a pattern lock. Once the pattern is setup, all data will be locked. If you forgot your pattern, reinstall the app but all data will be lost. To change the pattern, simply enter a new one.

View individual data by clicking on the specific tab. Quick search i also available. Remove individual data by clicking the remove button.

Add data by entering a name and details in the "Add" tab. The "Details" section can be anything, all HTML will be parsed.

!!! Important !!! Your saved data is not saved online, only on your local machine. To move it to another machine you have to save your data to a text file and import it into the new browser extension.
